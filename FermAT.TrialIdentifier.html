FermAT.TrialIdentifier module¶

class FermAT.TrialIdentifier.RunIdentifier¶

Bases: object

Carries information about the run through all the objects

Attributes

strainID (str) Strain name e.g. ‘MG1655 del(adh,pta)’
identifier1 (str, optional) First identifier, plasmid e.g. ‘pTrc99a’
identifier2 (str, optional) Second identifier, inducer e.g. ‘IPTG’
replicate (str) The replicate number, e.g. ‘1’,‘2’,‘3’,‘4’
time (float) The time of the data point, only relevant in TimePoint objects
analyte_name (str) The name of the titer, e.g. ‘OD600’,’Lactate’,’Ethanol’,’Glucose’
titerType (str) The type of titer, three acceptable values e.g. ‘biomass’,’substrate’,’product’
