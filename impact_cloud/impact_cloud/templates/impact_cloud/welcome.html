The Fermentation Analysis Toolbox

#} {##}

Impact helps scientists extract insight from experiments to characterize microbial physiology.

Expedite parsing raw data, inspecting for outliers and creating visualizations for web or publication, as well as storing, accessing and sharing data. It is best suited for experiments from medium to high-throughput.

IMPACT is open-source and available on GitHub to promote active development and collaboration.
